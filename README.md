# Defenda o Reino

Defenda o Reino é um jogo de estratégia desenvolvido em JavaScript, HTML e CSS. O objetivo do jogo é proteger seu reino contra invasores, utilizando diferentes classes e mecânicas de combate.

## Funcionalidades

- Sistema de combate entre personagens
- Diferentes classes de personagens
- Interface interativa em HTML e CSS

## Estrutura do Projeto

- `index.html`: Página principal do jogo
- `style.css`: Estilos visuais do jogo
- `app.js`: Lógica principal do jogo
- `classes.js`: Definição das classes dos personagens
- `combate.js`: Mecânicas de combate

## Como Jogar

1. Abra o arquivo `index.html` em seu navegador.
2. Siga as instruções na tela para iniciar o jogo.
3. Defenda seu reino utilizando as opções disponíveis.

## Requisitos

- Navegador moderno (Chrome, Firefox, Edge, etc.)
